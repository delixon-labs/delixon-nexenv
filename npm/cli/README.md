# nexenv

Workspace manager for developers. GUI + CLI.

Stop configuring. Start building.

## Install

```bash
npm install -g nexenv
```

## Usage

```bash
nexenv --version
nexenv --help
```

## Enterprise

```bash
nexenv license activate YOUR-KEY
```
